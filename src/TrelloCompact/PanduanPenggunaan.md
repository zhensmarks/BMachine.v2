# TrelloCompact - Panduan Penggunaan Lengkap

Selamat datang di **TrelloCompact**! Aplikasi ini dirancang untuk mempermudah Anda dalam melihat, mengatur, dan mengelola daftar kerja (Cards) dari Trello dengan tampilan yang lebih padat, cepat, dan ringan di Desktop Windows Anda.

Aplikasi ini sangat cocok bagi Anda yang butuh bekerja dengan banyak kartu Trello sekaligus tanpa harus bolak-balik membuka browser.

---

## 🚀 1. Persiapan Awal (Satu Kali Saja)

Sebelum bisa menggunakan TrelloCompact, Anda perlu menyambungkannya ke akun Trello Anda menggunakan API Key dan Token.

1. Buka aplikasi **TrelloCompact**. Jika ini pertama kalinya, Anda akan langsung disapa oleh layar "Pengaturan Awal" (Setup).
2. Anda harus mengisi **Trello API Key** dan **API Token**.
   - Untuk mendapatkannya, klik tombol link biru bertuliskan **"Dapatkan API Key di sini"** yang ada di layar.
   - Login ke akun Trello Anda di browser.
   - *Copy* (Salin) bagian "Key" dan *paste* (tempel) ke kolom **Trello API Key** di TrelloCompact.
   - Di halaman browser yang sama, klik link "Token", berikan izin (Allow), lalu copy teks panjang yang muncul dan paste ke kolom **API Token**.
3. Isi kolom **Nama User Anda / Display Name** (bebas) untuk membedakan Anda di dalam TrelloCompact.
4. Klik **Terapkan Pengaturan**.

Aplikasi kini sudah terhubung dengan Trello Anda!

---

## 📑 2. Menambahkan Tab (Daftar/List Trello)

TrelloCompact bekerja dengan sistem "Tab". Setiap Tab akan menampilkan isi dari satu buah List/Daftar yang ada di Trello.

1. Di layar utama bagian kiri atas, klik tombol **"+" (Tambah Tab)**.
2. Jendela pemilihan akan muncul. Aplikasi akan memuat semua Board (Papan) Trello Anda.
3. Pilih **Board** yang Anda inginkan dari menu *dropdown* pertama.
4. Setelah Board dipilih, pilih **List (Daftar)** yang ingin Anda pantau dari *dropdown* kedua.
5. Beri nama yang unik pada **Nama Tab Tambahan** agar Anda mudah mengenalinya.
6. Pilih **Warna Tab** agar Tab Anda terlihat lebih cantik dan mudah dibedakan.
7. Klik **+ TAMBAH TAB INI**. Tab baru akan muncul di menu paling atas!

Anda bisa membuat banyak Tab untuk memantau List Trello yang berbeda-beda secara bebarengan.

---

## 🎯 3. Cara Menggunakan Fitur-Fiturnya

TrelloCompact memiliki banyak jalan pintas dan fitur hebat untuk mempercepat kerja Anda.

### A. Melihat Detail Kartu
- **Klik sekali** pada kartu manapun di dalam daftar untuk melihat secara detail (anggota yang ditugaskan, tanggal tenggat waktu, warna label, hingga deskripsinya).

### B. Membuka Menu Kartu (Komentar, Checklist, Lampiran)
Saat Anda sudah mengklik sebuah kartu:
- Klik **Komentar (💬)** untuk membaca atau membalas komentar.
- Klik **Checklist (☑)** untuk melihat, mencentang, atau menambah daftar To-Do.
- Klik **Lampiran (📎)** untuk melihat file yang terlampir. Anda juga bisa mengklik tombol "Buka" untuk langsung melihat file lampirannya di browser Anda.

### C. Menambahkan Kartu Cepat Lewat Link/URL
Jika Anda menemukan link Trello di tempat lain dan ingin memasukkannya ke List Anda sekarang:
1. Copy link URL kartu Trello tersebut (misal dari chat WA atau Email).
2. Di pojok kiri bawah TrelloCompact ada tulisan **"Tempel Link URL Trello Di Sini :"**
3. Paste link tersebut ke dalam kotak teks dan klik tombol **"+"**.
4. Kartu akan langsung otomatis berpindah ke daftar Anda saat ini!

### D. Fitur Seleksi Massal (Batch Actions)
Ingin memindahkan atau mengcopy banyak kartu sekaligus?
1. **Centang** kotak kecil di sebelah kiri kartu-kartu yang ingin Anda proses. (Anda akan melihat tombol-tombol baru muncul di bagian atas daftar saat Anda mencentang kartu).
2. **Move Selected (Pindahkan)**: Klik ini untuk memindahkan semua kartu yang dicentang sekaligus ke Board atau List lain secara bersamaan!
3. **Copy Links**: Klik ini untuk meng-copy (menyalin) semua Link URL kartu yang dicentang secara bersamaan. Sangat berguna jika Anda ingin melaporkan kumpulan link tersebut ke atasan via chat.
4. **Unlink Selected**: Tombol merah ini *hanya* akan muncul untuk kartu yang barusan Anda isikan lewat "Link / URL" (lihat poin C). Jika Anda tidak jadi memasukkan kartu tersebut ke daftar, klik tombol ini untuk mengeluarkan mereka kembali.

### E. Fitur Notepad (Buku Catatan Mini)
Kadang Anda butuh tempat corat-coret sementara untuk masing-masing Tab.
- Di bagian atas, klik tombol **"📝 Notepad"**.
- Sebuah panel catatan akan muncul di sebelah kanan. Catatan ini **hanya untuk Anda** dan akan tersimpan setiap kali Anda mengetik. Catatan ini juga berbeda-beda antara satu Tab dengan Tab lainnya!

### F. Menyembunyikan Tab (Reorder)
- Jika Tab di atas terasa terlalu penuh, Anda punya menu **"Ubah List Tab"** (Tombol gear ⚙️ di samping tombol tambah tab). 
- Di sini Anda bisa menggeser susunan urutan Tab Anda, mengubah warna Tab yang sudah ada, atau menghapus Tab yang sudah tidak Anda butuhkan menggunakan ikon Tempat Sampah.

## 🛠️ Tanya Jawab Ringkas (FAQ)
- **Tanya**: Kenapa aplikasi bilang "Loading..." terus?
- **Jawab**: Pastikan komputer Anda terhubung ke internet. TrelloCompact langsung berkomunikasi dengan server Trello. Jika terputus saat pertama kali, dia akan mencoba mengambil data lama saat layar Anda mati menggunakan "mode Offline".

- **Tanya**: Kok saya centang banyak kartu, tapi tombol `Unlink Selected` yang merah tidak muncul?
- **Jawab**: Tombol merah itu dirancang secara spesial hanya untuk mencabut/mengembalikan kartu yang asalnya Anda tarik paksa lewat Link/URL (fitur C). Kartu Trello yang memang dari sananya sudah ada di List tersebut tidak bisa pakai tombol ini. Gunakan tombol `Move Selected` untuk memindahkannya ke tempat yang seharusnya.
